O que é este Site?

Este Site é um projeto desenvolvido por mim na disciplina de programação.

O intuito desde Site é facilitar (talvez nem tanto) a busca por livros, utilizando para este fim, a plataforma gratuita Google Books.

Todos os livros desta página são de domínio público, ou seja, tem acesso livre para todos.

Entretanto as imagens do mesmo não, logo, estão sujeitas a alterações.
